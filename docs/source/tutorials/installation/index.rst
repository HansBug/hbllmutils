Installation
===================

``hbllmutils`` is currently hosted on PyPI. It required python >= 3.7.

You can simply install hbllmutils from PyPI with the following command:

.. code:: shell

    pip install hbllmutils

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/hansbug/hbllmutils.git@main

of ``hbllmutils`` should be shown.
